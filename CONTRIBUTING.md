# Contributing to GET Learning Passport

Thank you for your interest in contributing to the GET Learning Passport! This project is open source under the MIT License as part of our commitment to building digital public goods.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Development Setup

```bash
git clone https://github.com/Clickshift-Founder/get-learning-passport.git
cd get-learning-passport
npm install
npx hardhat compile
npx hardhat test
```
